# at_hive_recovery

**at_hive_recovery** tool for recovering corrupted hive boxes.

Usage:

```
dart bin/main.dart <atsign> <dir_path_of_hive_box>
```

e.g 

```
dart bin/main.dart @alice /home/alice/storage/hive
```
